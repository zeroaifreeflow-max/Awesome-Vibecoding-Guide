# Tooling

> A minimal, efficient toolkit for profitable vibecoding.

## Philosophy

**Technique > Tools.** The LLM behind vibecoding doesn't matter as much as your methodology. This toolkit is:

- **Minimal** — Only what you need, nothing more
- **Cost-effective** — Budget-friendly, BYOK where possible
- **Stable** — Tools that won't change every week
- **Proven** — Used in real commercial projects

## BYOK Principle

**Bring Your Own Key** — All recommended tools support external API integration:
- Use your preferred providers
- Control your costs
- Switch models as needed
- Keep your data secure
