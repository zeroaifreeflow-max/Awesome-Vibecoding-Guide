# QA Checklist

Test every item before client review or launch.

---

## Functionality

- [ ] All navigation links work
- [ ] All buttons trigger correct actions
- [ ] Forms submit successfully
- [ ] Form validation shows errors correctly
- [ ] Confirmation messages display
- [ ] Email notifications received
- [ ] External links work and open in new tab
- [ ] Responsive on all screen sizes

## Design

- [ ] Matches approved design
- [ ] Consistent spacing throughout
- [ ] Fonts load correctly
- [ ] Colors match brand guidelines
- [ ] Hover states work on interactive elements
- [ ] No layout shifts or broken elements
- [ ] Images not stretched or pixelated

## Accessibility

- [ ] All images have alt text
- [ ] Keyboard navigation works
- [ ] Focus states visible
- [ ] Color contrast passes (4.5:1 min)
- [ ] Headings in correct order (H1 → H2 → H3)
- [ ] Form fields have labels
- [ ] Skip-to-content link works

## Performance

- [ ] Page loads in under 3 seconds
- [ ] Images compressed and optimized
- [ ] No large file downloads on page load
- [ ] No console errors or warnings

## Cross-Browser Testing

- [ ] Chrome (desktop)
- [ ] Safari (desktop)
- [ ] Firefox (desktop)
- [ ] Chrome (mobile)
- [ ] Safari (iOS)
- [ ] Edge

## Content

- [ ] No lorem ipsum or placeholder text
- [ ] Phone numbers and emails correct
- [ ] Address and location correct
- [ ] Business hours accurate
- [ ] Social links go to correct profiles
- [ ] Copyright year current
