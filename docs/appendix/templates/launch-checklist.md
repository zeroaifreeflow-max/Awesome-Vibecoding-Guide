# Website Launch Checklist

Complete before going live.

---

## Content

- [ ] All text proofread and spell-checked
- [ ] No placeholder text (lorem ipsum)
- [ ] Images optimized and compressed
- [ ] All images have alt text
- [ ] Contact information is correct
- [ ] Copyright year is current

## Technical

- [ ] All forms submit correctly
- [ ] Form confirmations/emails working
- [ ] 404 error page exists and looks good
- [ ] All internal links work
- [ ] All external links open in new tab
- [ ] robots.txt is configured
- [ ] No console errors in browser

## SEO

- [ ] Page titles set (unique per page)
- [ ] Meta descriptions written
- [ ] Open Graph image set
- [ ] Sitemap.xml generated
- [ ] Site submitted to Google Search Console

## Performance

- [ ] Lighthouse score 90+ (mobile)
- [ ] Images lazy-loaded
- [ ] Tested on mobile device
- [ ] Tested on tablet
- [ ] Page load under 3 seconds

## Legal

- [ ] Privacy policy page
- [ ] Terms of service (if applicable)
- [ ] Cookie notice (if required)
- [ ] Accessibility statement (if required)

## Analytics

- [ ] Google Analytics installed
- [ ] Analytics tracking verified
- [ ] Goals/conversions configured

## Domain & Hosting

- [ ] DNS configured correctly
- [ ] SSL certificate active (https)
- [ ] www redirects to non-www (or vice versa)
- [ ] Old site redirects set up (if redesign)
- [ ] Backups configured
