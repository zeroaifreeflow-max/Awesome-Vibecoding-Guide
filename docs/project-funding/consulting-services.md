# Consulting & Services Model

> Building a consulting business around vibecoding expertise.

## Overview

Consulting leverages your expertise for premium rates. This guide covers service offerings, pricing, and client acquisition for vibecoding consultants.

## Service Offerings

### Core Services

| Service | Description | Typical Duration | Price Range |
|---------|-------------|------------------|-------------|
| Vibecoding Workshop | Group training session | 1 day | $5,000-15,000 |
| Team Training | Ongoing skill building | 4-8 weeks | $10,000-25,000 |
| Implementation Support | Project assistance | Per project | $2,000-10,000 |
| Code Audit | Security/performance review | 1-2 weeks | $2,000-10,000 |
| Strategy Consultation | Roadmaps and planning | Per engagement | $500-2,000/session |

### Referral Program

**Referral Rewards:**
- 15% of first project value
- 5% of subsequent projects for 1 year

**How It Works:**
1. Client refers new client
2. New client books project
3. Referrer receives reward
4. Both get 10% off next project

## Engagement Templates

### Discovery Call Script

**Introduction (2 min):**
```
"Thanks for taking the time today. Before we dive in,
I want to understand your situation. Can you tell me
about [Company] and what prompted this conversation?"
```

**Discovery Questions (10 min):**
1. What's the current challenge?
2. What have you tried so far?
3. What would success look like?
4. What's your timeline?
5. What's your budget range?

**Solution Overview (5 min):**
```
"Based on what you've shared, it sounds like
[solution] could help. Here's how we've helped
similar clients..."
```

**Next Steps (3 min):**
```
"To move forward, I'd recommend [proposal].
Does that make sense? When could we schedule
a follow-up?"

[If yes:] "Great, I'll send over a proposal by [date]."
[If no:] "No problem. What's the best way to
stay in touch?"
```

### Proposal Template

**Structure:**
```
1. Executive Summary
2. Understanding of Situation
3. Proposed Solution
4. Deliverables
5. Timeline
6. Investment
7. Terms
8. Next Steps
```

### Contract Essentials

**Must-Have Clauses:**
- Scope of work
- Payment terms
- Timeline
- Revisions
- Cancellation
- Confidentiality
- Liability limit

**Sample Payment Terms:**
```
50% deposit upon signing
25% at midpoint
25% upon completion

Net 15 for invoices
1.5% monthly interest on late payments
```
