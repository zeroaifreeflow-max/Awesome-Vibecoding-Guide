# Scaling Quick Reference

## 5 Phases of Growth

Source: `docs/business-operations/scaling.md`

| Phase | Revenue/mo | Team | Key Focus | Exit Criteria |
| ----- | ---------- | ---- | --------- | ------------- |
| 1. Solo | $0-5k | Just you | Learn, deliver, portfolio | 3 months consistent |
| 2. Leveraged | $5-15k | + 1 contractor | Document systems, first hire | Systems documented |
| 3. Team | $15-30k | + PM + 2-3 contractors | Delegate, QC | Revenue without daily involvement |
| 4. Agency | $30-100k | 15-20 people | Leadership team | Self-running operations |
| 5. Product | $100k+ | 10+ | SaaS, passive income | Product-market fit |

## Website Growth Timeline

Source: `docs/02-websites-playbook/scaling.md`

| Month | Target | Focus |
| ----- | ------ | ----- |
| 1-3 | 2 sites/mo ($600-1,400) | Build portfolio, refine process |
| 4-6 | 4 sites/mo ($1,600-2,800) | Referrals start, faster delivery |
| 7-12 | 6-10 sites/mo ($3,600-7,000) | Systems, possibly first hire |

## 5 Scaling Strategies

1. **Vertical**: General $1,200 → Specialist $3,500 (niche down)
2. **Horizontal**: Add service lines (websites → automations → SaaS)
3. **Product**: Templates, courses, tools (productized services)
4. **Partnership**: Referral, white-label, joint ventures
5. **Team**: Hire and delegate

## Financial Formulas

```
Billable Hours = Total Hours x 0.55
Minimum Rate = Monthly Target / Billable Hours
Profit Margin Target = 40-60%
Cash Reserve = 3-6 months (Phase 1-2), 6-12 months (Phase 3+)
```
