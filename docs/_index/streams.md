# Revenue Streams Quick Reference

## Client Income Streams (docs/01-money-map/)

### Stream 1: Local Business Websites

- **Price:** $300-1,000/site
- **Recurring:** $25-100/mo maintenance
- **Time to first $:** 1-2 weeks
- **Difficulty:** Beginner
- **Playbook:** `docs/02-websites-playbook/`
- **Pricing detail:** `docs/02-websites-playbook/pricing-economics.md`

### Stream 2: AI Automations

- **Price:** $500-3,000 setup
- **Recurring:** $50-300/mo retainer
- **Time to first $:** 2-4 weeks
- **Difficulty:** Intermediate
- **Playbook:** `docs/03-automations-playbook/`
- **Pricing detail:** `docs/03-automations-playbook/pricing-automations.md`

### Stream 3: Client-First Micro-SaaS

- **Price:** $500-2,000 setup
- **Recurring:** $50-500/mo subscription
- **Time to first $:** 1-2 weeks (with existing clients)
- **Difficulty:** Advanced
- **Playbook:** `docs/04-micro-saas-playbook/`
- **Pricing detail:** `docs/04-micro-saas-playbook/scoping-pricing.md`

## Key Formula

```
Client Value = (Time Saved x Hourly Rate) + Revenue Increase + Cost Reduction
Your Price = 10-20% of Client Value
```

## Progressive Path

Websites (beginner) → Automations (intermediate) → Micro-SaaS (advanced)

Each stream builds on skills and clients from the previous one.
