# IDE

> Your development environment for vibecoding.

## Recommended

### [Zed](zed.md)
**Primary IDE** — Fast, modern, BYOK-friendly.

- Native speed (Rust-based)
- Built-in AI features with BYOK support
- Multi-cursor, collaborative editing
- Clean, distraction-free interface

## Why Zed?

| Feature | Benefit |
|---------|---------|
| **Speed** | No Electron lag |
| **BYOK AI** | Connect your own providers |
| **Modern** | Built for current workflows |
| **Free** | No subscription needed |

## Alternatives

If you need something else:
- **VS Code** — Familiar, extensive extensions (but slower)
- **Cursor** — AI-focused (but subscription-based)
- **Terminal-based** — Neovim/Helix for minimal setup

For most vibecoding work, Zed provides the best balance of speed, features, and cost.
